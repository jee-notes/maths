Arithmetic Progression (A.P.)

Linear discrete functions and constant sequential growth

Definition of an Arithmetic Progression

An Arithmetic Progression (A.P.) is a sequence of numbers in which the difference between any two consecutive terms is a constant. This constant difference is universally denoted as the common difference (\(d\)).

The Core Constraint

A sequence \(T_1, T_2, T_3, \dots, T_n\) is an A.P. if and only if: \[T_{k+1} - T_k = d \quad \text{for all } k \ge 1\] where \(d\) is a constant independent of \(k\).

If the first term of the A.P. is designated as \(a\), the sequence unfolds as: \[a, \ a + d, \ a + 2d, \ a + 3d, \dots\]

The General Term (\(T_n\))

To find any arbitrary term in the sequence without calculating all preceding terms, we construct a formula for the \(n^{\text{th}}\) term (\(T_n\)).

Observing the pattern of the sequence:

  • \(T_1 = a = a + (1 - 1)d\)
  • \(T_2 = a + d = a + (2 - 1)d\)
  • \(T_3 = a + 2d = a + (3 - 1)d\)
By inductive reasoning, the \(n^{\text{th}}\) term is given by:
\[T_n = a + (n - 1)d\]

Geometrical Interpretation

An Arithmetic Progression is the discrete equivalent of a continuous **linear function**. If you plot the term value \(T_n\) on the y-axis against the term index \(n\) on the x-axis, the points will lie perfectly on a straight line.

In this linear mapping:
  • The common difference \(d\) acts as the slope or gradient.
  • The first term \(a\) sets the initial starting height.
Because it is a linear function of \(n\), any sequence whose \(n^{\text{th}}\) term is of the form \(T_n = pn + q\) is guaranteed to be an A.P. with common difference \(p\).

Sum of \(n\) Terms of an A.P. (\(S_n\))

Let \(S_n\) denote the sum of the first \(n\) terms of an A.P.: \[S_n = a + (a+d) + (a+2d) + \dots + [a + (n-1)d]\] According to mathematical legend, Carl Friedrich Gauss derived this formula as a child by reversing the sequence and adding it to itself: \[S_n = T_1 + T_2 + \dots + T_{n-1} + T_n\] \[S_n = T_n + T_{n-1} + \dots + T_2 + T_1\] Adding the two equations vertically, each pair of terms \((T_k + T_{n-k+1})\) sums to exactly the same constant value \((T_1 + T_n)\): \[2S_n = (T_1 + T_n) + (T_1 + T_n) + \dots \text{ (}n\text{ times)}\] \[2S_n = n(T_1 + T_n)\] This yields the first sum formula:

\[S_n = \frac{n}{2} [T_1 + T_n]\]
Substitute \(T_1 = a\) and \(T_n = a + (n-1)d\) into the formula to obtain the general sum formula:
\[S_n = \frac{n}{2} [2a + (n - 1)d]\]

Symmetry of Sums

The sum of an A.P. is a **quadratic function** of \(n\) with no constant term. Any sequence whose sum to \(n\) terms is of the form \(S_n = Pn^2 + Qn\) is an A.P., and its common difference is exactly \(2P\) (twice the coefficient of \(n^2\)).

Arithmetic Mean (A.M.)

When three numbers \(a, b, c\) are in Arithmetic Progression, the middle term \(b\) is called the **Arithmetic Mean (A.M.)** of the other two.

Since they are in A.P., the common difference must be equal: \[b - a = c - b \implies 2b = a + c\]

\[b = \frac{a + c}{2}\]

Insertion of \(n\) Arithmetic Means

If we wish to insert \(n\) Arithmetic Means (\(A_1, A_2, \dots, A_n\)) between two given numbers \(a\) and \(b\), the resulting sequence: \[a, A_1, A_2, \dots, A_n, b\] must form an A.P.

This sequence has a total of \((n + 2)\) terms. The last term \(b\) is the \((n+2)^{\text{th}}\) term: \[T_{n+2} = a + (n+2-1)d = b\] \[a + (n+1)d = b \implies d = \frac{b - a}{n + 1}\] Once \(d\) is known, any inserted mean can be calculated: \[A_k = a + kd \quad \text{for } 1 \le k \le n\]

Core Properties of an A.P.

  • If a constant is added, subtracted, multiplied, or divided across all terms of an A.P., the resulting sequence remains an A.P.
  • In a finite A.P., the sum of the terms equidistant from the beginning and the end is always constant and equal to the sum of the first and last terms: \[T_k + T_{n-k+1} = T_1 + T_n\]
  • If you need to assume **three** terms in A.P. for a problem, choose: \(a-d, \ a, \ a+d\).
  • If you need to assume **four** terms in A.P., choose: \(a-3d, \ a-d, \ a+d, \ a+3d\). (Note that the common difference here is \(2d\)).

Solved Examples

Example 1: Finding an A.P. from the Sum of \(n\) terms

Problem: The sum of the first \(n\) terms of a sequence is given by \(S_n = 3n^2 + 5n\). Find the \(n^{\text{th}}\) term and verify it is an A.P.
Solution: To find the \(n^{\text{th}}\) term (\(T_n\)) from the sum \(S_n\), we use the fundamental identity: \[T_n = S_n - S_{n-1}\] We are given \(S_n = 3n^2 + 5n\). Let's calculate \(S_{n-1}\): \[S_{n-1} = 3(n-1)^2 + 5(n-1)\] \[S_{n-1} = 3(n^2 - 2n + 1) + 5n - 5\] \[S_{n-1} = 3n^2 - 6n + 3 + 5n - 5\] \[S_{n-1} = 3n^2 - n - 2\] Now, calculate \(T_n\): \[T_n = (3n^2 + 5n) - (3n^2 - n - 2)\] \[T_n = 3n^2 + 5n - 3n^2 + n + 2\] \[T_n = 6n + 2\] Since \(T_n = 6n + 2\) is a linear polynomial in \(n\), the sequence is an Arithmetic Progression. The common difference is the coefficient of \(n\), which is \(6\).

Shortcut Check: Using the insight box rule, for \(S_n = Pn^2 + Qn\), the common difference is \(2P = 2(3) = 6\). Matches perfectly!

Example 2: Equidistant Terms Property

Problem: In an A.P. consisting of 11 terms, the sum of the 1st, 5th, 7th, and 11th terms is 48. Find the sum of all 11 terms of the A.P.
Solution: We are given an A.P. with \(n = 11\). The sum of the specified terms is: \[T_1 + T_5 + T_7 + T_{11} = 48\] By the property of equidistant terms, the sum of terms equidistant from the beginning and end is constant. Notice the indices:
  • \(1 + 11 = 12\)
  • \(5 + 7 = 12\)
Therefore, the pairs \((T_1, T_{11})\) and \((T_5, T_7)\) are equidistant from the ends, meaning: \[T_1 + T_{11} = T_5 + T_7\] Let this constant pair sum be \(K\). Then: \[K + K = 48 \implies 2K = 48 \implies K = 24\] Thus, \(T_1 + T_{11} = 24\).

We need to find the sum of all 11 terms (\(S_{11}\)). Using the formula based on the first and last terms: \[S_n = \frac{n}{2}(T_1 + T_n)\] Substitute \(n = 11\) and \((T_1 + T_{11}) = 24\): \[S_{11} = \frac{11}{2} (24) = 11 \times 12 = 132\] Conclusion: The sum of the 11 terms is 132.
