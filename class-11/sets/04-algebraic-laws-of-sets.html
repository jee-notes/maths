Algebraic Laws of Sets

The formal calculus of set relationships and algebraic transformations.

Fundamental Laws of Set Algebra

Just as numbers obey rules like commutativity and distributivity under addition and multiplication, sets obey a highly structured system of algebraic laws under union (\(\cup\)), intersection (\(\cap\)), and complementation (\('\)).

For any sets \(A\), \(B\), and \(C\) subset of Universal set \(U\), the following algebraic properties hold:

The Axiomatic Laws of Sets

  • Idempotent Laws: \[A \cup A = A \quad \text{and} \quad A \cap A = A\] (Operating a set with itself yields no change.)
  • Identity Laws: \[A \cup \emptyset = A \quad \text{and} \quad A \cap U = A\] (Empty set \(\emptyset\) is the identity element for union; Universal set \(U\) is the identity for intersection.)
  • Dominance / Boundedness Laws: \[A \cup U = U \quad \text{and} \quad A \cap \emptyset = \emptyset\] (Universal set dominates union; empty set dominates intersection.)
  • Commutative Laws: \[A \cup B = B \cup A \quad \text{and} \quad A \cap B = B \cap A\] (The order of binary operations is irrelevant.)
  • Associative Laws: \[(A \cup B) \cup C = A \cup (B \cup C) \quad \text{and} \quad (A \cap B) \cap C = A \cap (B \cap C)\] (Grouping of operations of the same kind is irrelevant.)
  • Distributive Laws:
    • Union distributes over intersection: \[A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\]
    • Intersection distributes over union: \[A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\]

Note on Distributivity

Be highly careful! In ordinary numbers, multiplication distributes over addition (\(a \cdot (b+c) = ab + ac\)), but addition does not distribute over multiplication (\(a + (b \cdot c) \neq (a+b) \cdot (a+c)\)).

In sets, however, symmetry is perfect: **both** operations distribute over each other! This is a core feature of Boolean algebras.

De Morgan's Laws and Proofs

Augustus De Morgan formulated two revolutionary laws that describe how complementation acts as a duality operator, exchanging union and intersection.

De Morgan's Statements

  1. First Law (Complement of Union): \[(A \cup B)' = A' \cap B'\] (The complement of a union is the intersection of the complements.)
  2. Second Law (Complement of Intersection): \[(A \cap B)' = A' \cup B'\] (The complement of an intersection is the union of the complements.)

Rigorously Proving the First Law: \((A \cup B)' = A' \cap B'\)

To prove this identity without relying on the suggestive visuals of Venn diagrams, we must utilize formal element logic and establish **double containment** (\(X \subseteq Y\) and \(Y \subseteq X\)).

Part 1: Prove \((A \cup B)' \subseteq A' \cap B'\)

Let \(x\) be an arbitrary element belonging to the left-hand set: \[x \in (A \cup B)'\] By definition of a complement: \[x \notin (A \cup B)\] If \(x\) does not belong to the union of \(A\) and \(B\), it means it cannot belong to either set: \[x \notin A \quad \text{and} \quad x \notin B\] By definition of complements: \[x \in A' \quad \text{and} \quad x \in B'\] By definition of intersections: \[x \in A' \cap B'\] Since any \(x \in (A \cup B)'\) must lie in \(A' \cap B'\), we have proved: \[(A \cup B)' \subseteq A' \cap B' \quad \text{--- (Equation 1)}\]

Part 2: Prove \(A' \cap B' \subseteq (A \cup B)'\)

Let \(y\) be an arbitrary element belonging to the right-hand set: \[y \in A' \cap B'\] By definition of intersection: \[y \in A' \quad \text{and} \quad y \in B'\] By definition of complements: \[y \notin A \quad \text{and} \notin B\] If \(y\) is not in \(A\) and is also not in \(B\), then \(y\) cannot possibly belong to their union: \[y \notin (A \cup B)\] By definition of complements: \[y \in (A \cup B)'\] Since any \(y \in A' \cap B'\) must lie in \((A \cup B)'\), we have proved: \[A' \cap B' \subseteq (A \cup B)' \quad \text{--- (Equation 2)}\]

Conclusion:

From Equation 1 and Equation 2, we conclude: \[(A \cup B)' = A' \cap B'\]

Algebraic Proofs of Set Identities

Instead of starting from element-wise containment for every single new identity, we can systematically leverage already-proven fundamental laws to simplify expressions. Let us study a common identity proof:

Proving: \(A - (B \cup C) = (A - B) \cap (A - C)\)

We will simplify the left-hand side (LHS) step-by-step:
  1. First, recall the fundamental set difference identity: \[X - Y \equiv X \cap Y'\] Applying this to the LHS: \[A - (B \cup C) = A \cap (B \cup C)'\]
  2. Apply De Morgan's First Law to the term \((B \cup C)'\): \[A \cap (B \cup C)' = A \cap (B' \cap C')\]
  3. Use the Idempotent Law to write \(A\) as \(A \cap A\): \[A \cap (B' \cap C') = (A \cap A) \cap (B' \cap C')\]
  4. Using the Associative and Commutative Laws of intersection, rearrange the grouping: \[(A \cap A) \cap (B' \cap C') = (A \cap B') \cap (A \cap C')\]
  5. Convert back to set difference notation using \(X \cap Y' = X - Y\): \[(A \cap B') \cap (A \cap C') = (A - B) \cap (A - C)\]
Thus, the left-hand side is algebraically transformed into the right-hand side: \[A - (B \cup C) = (A - B) \cap (A - C)\]

Solved Examples

Example 1: Simplification using Reverse Distributivity

Problem: Simplify the expression: \(E = (A \cup B) \cap (A \cup B')\).
Solution: Observe that \(A \cup\) is common to both bracketed terms. We can apply the Distributive Law in reverse (factoring out the union operator): \[(A \cup B) \cap (A \cup B') = A \cup (B \cap B')\] Recall the complement law: a set and its complement have absolutely no elements in common: \[B \cap B' = \emptyset\] Substitute this back: \[E = A \cup \emptyset\] By the Identity Law of union, union with an empty set leaves a set unchanged: \[E = A\] Thus, the simplified expression is **\(A\)**.

Example 2: Collapsing Difference Unions

Problem: Prove algebraically that \((A \cap B) \cup (A - B) = A\).
Solution: Express the set difference using intersection and complement: \[LHS = (A \cap B) \cup (A \cap B')\] Apply the distributive law in reverse (factoring out the intersection operator \(A \cap\)): \[LHS = A \cap (B \cup B')\] Recall the complement law: the union of a set and its complement constitutes the entire universal set: \[B \cup B' = U\] Substitute this back: \[LHS = A \cap U\] By the Identity Law of intersection, intersecting with the Universal set yields the set itself: \[LHS = A = RHS\] Hence, the identity is proved!

Example 3: Tricky Distributive Simplification

Problem: Simplify the complex set expression: \(X = A \cap (A \cup B)'\).
Solution: Let's apply De Morgan's Law to the complement term: \[(A \cup B)' = A' \cap B'\] Substitute this back: \[X = A \cap (A' \cap B')\] Apply the Associative Law of intersection to regroup the elements: \[X = (A \cap A') \cap B'\] Recall that \(A \cap A' = \emptyset\): \[X = \emptyset \cap B'\] Since the intersection of the empty set with any other set is strictly empty (Dominance Law): \[X = \emptyset\] Thus, the expression simplifies to **\(\emptyset\)** (the empty set).
