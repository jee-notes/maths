Introduction and Representation of Sets

The absolute baseline of structural mathematical vocabulary.

Definition of a Set

In naive set theory, a set is defined as a well-defined collection of distinct objects.

Let us break this definition down into its two crucial logical pillars:

1. Well-Defined Criterion

A collection is well-defined if there exists a clear, objective rule or condition that allows us to determine, with absolute certainty, whether any given object belongs to the collection or not.

There must be no subjective judgment, ambiguity, or individual bias involved.
  • Well-Defined (Is a Set): "The collection of all vowels in the English alphabet." (Objective. Any letter is either in \(\{a, e, i, o, u\}\) or it is not.)
  • Not Well-Defined (Not a Set): "The collection of the five best cricketers in India." (Subjective. The term "best" depends on personal criteria, opinions, and stats used.)

2. Distinct Objects

The objects in the collection must be **distinct**, meaning that no two elements can be identical. If an element is repeated in a collection, it is logically counted only once. For example, the collection of letters in the word "SCHOOL" is represented as \(\{S, C, H, O, L\}\), where the duplicate 'O' is collapsed.

Notation and Terminology:

  • Sets are conventionally denoted by capital letters: \(A, B, C, X, Y, Z\).
  • The objects belonging to a set are called its elements or members, usually denoted by lowercase letters: \(a, b, c, x, y\).
  • If an object \(x\) is an element of set \(A\), we write \(x \in A\) (read as "\(x\) belongs to \(A\)" or "\(x\) is an element of \(A\)").
  • If \(x\) is not an element of \(A\), we write \(x \notin A\) (read as "\(x\) does not belong to \(A\)").

Philosophical Insight: Russell's Paradox

By defining a set simply as "any well-defined collection," early mathematicians ran into severe logical contractions. The most famous is Russell's Paradox: Let \(R\) be the set of all sets that do not contain themselves. Does \(R\) contain itself?

If it does, then by definition it shouldn't. If it doesn't, then by definition it must! To resolve this paradox, modern mathematics utilizes axiomatic systems (such as Zermelo-Fraenkel set theory) which rule out such self-referential paradoxes. For JEE purposes, however, Cantor's intuitive, naive set theory is perfectly sufficient.

Standard Number Sets in Set Notation

Mathematics employs standard, universal symbols to denote the fundamental sets of numbers. You must memorize these notations as they are used constantly in JEE question prompts to restrict variables:

  • \(\mathbb{N}\): The set of all natural numbers (counting numbers) \(= \{1, 2, 3, 4, \dots\}\). Note that \(0 \notin \mathbb{N}\).
  • \(\mathbb{Z}\) or \(\mathbb{I}\): The set of all integers \(= \{\dots, -3, -2, -1, 0, 1, 2, 3, \dots\}\).
  • \(\mathbb{Z}^+\) or \(\mathbb{I}^+\): The set of all positive integers \(= \{1, 2, 3, \dots\}\) (identical to \(\mathbb{N}\)).
  • \(\mathbb{Z}^-\): The set of all negative integers \(= \{-1, -2, -3, \dots\}\).
  • \(\mathbb{Q}\): The set of all rational numbers \(= \{ \frac{p}{q} \mid p, q \in \mathbb{Z}, q \neq 0 \}\).
  • \(\mathbb{R}\): The set of all real numbers, representing the entire continuous real number line.
  • \(\mathbb{R}^+\): The set of positive real numbers \((0, \infty)\).
  • \(\mathbb{C}\): The set of all complex numbers \(= \{a + ib \mid a, b \in \mathbb{R}, i = \sqrt{-1}\}\).

These number sets form a strict nested hierarchical relationship: \[\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}\]

Roster or Tabular Form

In the **Roster Form**, a set is represented by listing all its elements explicitly, separated by commas, and enclosing them within curly brackets \(\{ \}\).

For example, the set of all positive even integers less than \(10\) is written as: \[A = \{2, 4, 6, 8\}\]

Core Rules of Roster Form

  1. Order is Irrelevant: The order in which elements are listed does not change the set. \[\{1, 2, 3\} \equiv \{3, 1, 2\} \equiv \{2, 3, 1\}\]
  2. Repetition is Redundant: All elements must be distinct. If an element is written multiple times, it is collapsed into a single representation. \[\{a, a, b, b, c\} \equiv \{a, b, c\}\]
  3. Ellipsis (\(\dots\)) for Infinite Sets: If a set has infinitely many elements or is too large to write fully, we establish a clear pattern with the first few elements and use dots to indicate continuation. \[\mathbb{N} = \{1, 2, 3, \dots\}\]

Set-Builder Form

Listing every element is impossible for uncountably infinite sets (like real numbers in an interval) and highly tedious for large finite sets. In such cases, we use the **Set-Builder Form**.

In Set-Builder Form, all elements of the set possess a single, common property that does not belong to any object outside the set.

The general syntax of Set-Builder Form is: \[A = \{ x \mid P(x) \} \quad \text{or} \quad A = \{ x : P(x) \}\]

Where:

  • \(x\) represents a typical element of the set.
  • The colon \(:\) or vertical bar \(\mid\) stands for the phrase "such that".
  • \(P(x)\) is the characteristic property or condition that the variable \(x\) must satisfy to qualify as a member of set \(A\).

For example, the set \(A = \{2, 4, 6, 8\}\) can be expressed in set-builder form as: \[A = \{ x \in \mathbb{N} \mid x \text{ is even and } x < 10 \}\]

Roster Form vs. Set-Builder Form Comparison

Roster Form Set-Builder Form Mathematical Description
\(\{a, e, i, o, u\}\) \(\{x \mid x \text{ is a vowel in English alphabet}\}\) Vowels in English
\(\{1, 3, 5, 7, 9\}\) \(\{x \in \mathbb{N} \mid x \text{ is odd and } x \le 9\}\) First five odd natural numbers
\(\{1, 4, 9, 16, 25\}\) \(\{x^2 \mid x \in \mathbb{N}, x \le 5\}\) Perfect squares up to 25
\(\{-3, 3\}\) \(\{x \in \mathbb{Z} \mid x^2 = 9\}\) Roots of the equation \(x^2 = 9\)

Solved Examples

Example 1: Solving Equations and Converting to Roster Form

Problem: Write the set \(A = \{ x \in \mathbb{R} \mid x^3 - 6x^2 + 11x - 6 = 0 \}\) in Roster form.
Solution: The set contains the real roots of the cubic polynomial equation: \[x^3 - 6x^2 + 11x - 6 = 0\] Let's factorize the cubic equation. By trial, we substitute \(x = 1\): \[(1)^3 - 6(1)^2 + 11(1) - 6 = 1 - 6 + 11 - 6 = 0\] Since \(x = 1\) is a root, \((x - 1)\) is a factor. Divide the polynomial by \((x - 1)\): \[x^3 - 6x^2 + 11x - 6 = (x - 1)(x^2 - 5x + 6)\] Now, factorize the quadratic term \(x^2 - 5x + 6\): \[x^2 - 5x + 6 = (x - 2)(x - 3)\] Thus, the factored cubic equation is: \[(x - 1)(x - 2)(x - 3) = 0\] The roots of the equation are \(x = 1\), \(x = 2\), and \(x = 3\). Since all three roots are real numbers, the set in Roster form is: \[A = \{1, 2, 3\}\]

Example 2: Analyzing Patterns and Representing in Set-Builder Form

Problem: Write the set \(B = \left\{ \frac{1}{2}, \frac{2}{5}, \frac{3}{10}, \frac{4}{17}, \frac{5}{26} \right\}\) in Set-Builder form.
Solution: Let's analyze the relationship between the numerator and denominator of each element in the set:
  • First element: \(\frac{1}{2} = \frac{1}{1^2 + 1}\)
  • Second element: \(\frac{2}{5} = \frac{2}{2^2 + 1}\)
  • Third element: \(\frac{3}{10} = \frac{3}{3^2 + 1}\)
  • Fourth element: \(\frac{4}{17} = \frac{4}{4^2 + 1}\)
  • Fifth element: \(\frac{5}{26} = \frac{5}{5^2 + 1}\)
Clearly, if the numerator is \(n\), the denominator is of the form \(n^2 + 1\). The numerator \(n\) is a natural number running from \(1\) to \(5\) inclusive.

Thus, we can express the set in Set-Builder form as: \[B = \left\{ \frac{n}{n^2 + 1} \mid n \in \mathbb{N}, n \le 5 \right\}\]

Example 3: Tricky Integer Constraints

Problem: Express the set \(C = \{ x \in \mathbb{Z} \mid 4x^2 - 19x - 5 < 0 \}\) in Roster form.
Solution: First, solve the quadratic inequality: \[4x^2 - 19x - 5 < 0\] Factorize the quadratic expression: \[4x^2 - 20x + x - 5 < 0 \implies 4x(x - 5) + 1(x - 5) < 0 \implies (4x + 1)(x - 5) < 0\] Using the wavy curve method (interval testing), the product is negative when \(x\) lies between the roots of the equation: \[-\frac{1}{4} < x < 5\] The set builder condition specifies that \(x\) must be an integer (\(x \in \mathbb{Z}\)). We must list all integers that satisfy this inequality: The integers in the open interval \(\left(-\frac{1}{4}, 5\right)\) are: \[x = 0, 1, 2, 3, 4\] Thus, in Roster form, the set is: \[C = \{0, 1, 2, 3, 4\}\]
