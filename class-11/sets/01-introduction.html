Sets

The absolute foundation of mathematics.

Maths is the Language of the World

If humans are the most intelligent beings on Earth, then the greatest tool ever created by humans is not fire or the wheel. It is mathematics.

Initially, people believed mathematics came directly from the physical world. Natural numbers emerged from counting objects. Geometry emerged from measuring land and studying space.

But eventually, mathematics evolved beyond physical reality. People realised that mathematics does not fundamentally depend on the real world. It depends on definitions and logic.

In mathematics, statements are true because they logically follow from previously accepted definitions and rules.

For example:

If all parrots are monkeys, and all monkeys are dogs, then logically, all parrots are dogs.

The statement may be physically absurd, but the logic itself is perfectly valid.

Similarly,

\(1 + 2 = 3\)

is true because of how numbers and operations are defined.

Modern mathematics is therefore a massive logical structure built from definitions. And the most fundamental language used to build that structure is the language of sets.

Why Fundamental Concepts are Necessary

When building any logical system, we cannot define every single word using other words.

If we keep defining terms forever using other terms, we enter an infinite chain of definitions — an infinite regress.

Consider a non-mathematical example: We can define a student as someone who studies. We can then define studying as acquiring knowledge, and knowledge as verified information.

But if we continue this process endlessly, the system eventually collapses into either infinite regression or circular self-reference.

To avoid infinite regress, every mathematical system must begin with a few foundational concepts that are accepted without formal definition.

These are called primitive notions.

Primitive notions are not understood through definitions. They are understood through the rules (axioms) that govern how they interact with one another.

Infinite Regress:
Infinite regress is simply a never-ending chain of definitions — like a bottomless pit of vocabulary with no fixed baseline.
In our example, we define information using another word, then define that word using still another word, and so on forever.
To prevent this, we deliberately stop at some foundational point and accept a few concepts without further definition because their meaning is universally intuitive through shared human experience.

Mathematics also relies heavily on abstraction.

Different physical objects may be completely unique in the real world, yet mathematics strips away their physical properties to expose common structural patterns.

For instance, two apples may differ in color, shape, texture, and weight. But mathematics ignores those physical differences and focuses only on the abstract quantity: \(2\)

This ability to ignore physical details and focus only on structure is one of the core powers of mathematics.

Why Sets are Fundamental

Modern mathematics is most commonly built on the language of sets.

Everything is a Set:
Think of mathematics like grammar.
In language, every entity or object behaves like a noun.
In mathematics, every entity behaves like a set.
Operations such as addition, differentiation, integration, or transformations behave like verbs — they are rules that act upon those sets.

Before we can understand how mathematical systems calculate or operate, we must first understand what mathematical entities themselves are structurally built from.

Under standard set theory, every advanced mathematical object — whether it is a number, a function, a geometric space, or a complex structure — can ultimately be represented using sets.

This does not mean mathematical objects physically “look like” sets in practice. It means they can be rigorously encoded using set-theoretic structures.

1. The Structural Reduction of Numbers

We usually treat numbers as fundamental constants.

However, standard set theory constructs the entire number system purely from the empty set: \(\emptyset\) which is simply a set containing zero elements.

  • 0 is defined as the empty set: \(\emptyset\)
  • 1 is defined as the set containing 0: \(\{\emptyset\}\)
  • 2 is defined as the set containing 0 and 1: \(\{\emptyset, \{\emptyset\}\}\)
  • 3 is defined as the set containing 0, 1, and 2: \(\{\emptyset, \{\emptyset\}, \{\emptyset, \{\emptyset\}\}\}\)

By repeatedly applying this exact construction rule, every natural number can be built purely from nested set structures.

Once natural numbers are constructed, set-theoretic methods can then be used to construct integers, rational numbers, real numbers, and complex numbers.

2. The Structural Reduction of Functions

In calculus and algebra, a function such as \(f(x) = y\) is often visualized as a machine that maps an input to an output.

But if we strip away the operational intuition and ask what a function fundamentally is, the structure again reduces back to sets.

  • A function is formally defined as a specific type of relation.
  • A relation is defined as a subset of a Cartesian product.
  • A Cartesian product is a set consisting entirely of ordered pairs \((x,y)\).
  • An ordered pair \((x,y)\) itself can be defined purely as a set structure: \(\{\{x\},\{x,y\}\}\)

You do not need to memorize this construction. The important insight is that even functions and ordered pairs can ultimately be encoded using sets.

Without the primitive language of sets, functions cannot be rigorously defined.

3. The Structural Reduction of Geometry and Spaces

In coordinate geometry, a line or curve is not fundamentally a physical mark on paper.

It is an infinite set of ordered pairs \((x,y)\) satisfying a particular equation.

A geometric figure is therefore fundamentally a set of points.

Similarly, mathematical spaces such as Vector Spaces, Euclidean Spaces, or Metric Spaces are built from:

  • An underlying set of abstract objects
  • A collection of rules or axioms governing their interaction

Modern geometry is therefore not merely the study of shapes, but the study of structured sets.

The Core Structural Paradigm

The architecture of modern mathematics follows a remarkably consistent structural pattern:

  1. The Raw Material:
    Begin with an abstract set of objects.
  2. The Structural Rules:
    Impose operations, relations, or axioms on that set.
  3. The Emergent Field:
    A complete branch of mathematics emerges from that structure.
Starting Set Imposed Structure / Rules Emergent Field
A set of abstract objects Algebraic operations satisfying group, ring, or field axioms Algebra
A set of abstract points Rules defining distance, continuity, or boundaries Geometry
A set of possible outcomes Probability rules assigning numerical weights Probability

If the concept of sets is removed, it is not merely one topic in algebra that disappears.

The structural foundations of algebra, calculus, geometry, probability, and mathematical logic all collapse together.

The set is therefore the primitive language through which modern mathematics is structurally defined.

The Ultimate Building Block:
In Chemistry, the Mole Concept is fundamental because it establishes that all physical matter is ultimately built from atoms and molecules.
Set theory plays a very similar foundational role in Mathematics.
Most mathematical objects and operations can ultimately be represented using set-theoretic structures.
Even in modern computing, every dataset, application, image, video, and algorithm ultimately reduces down to structured patterns of 1s and 0s.
