Types of Sets and Subsets

Understanding the hierarchy, boundaries, and internal spaces of set structures.

Types of Sets

To build operations on sets, we must first classify them based on their occupancy and cardinality:

1. Empty Set (Null Set / Void Set)

A set which contains no elements is called the empty set. It is denoted by the Greek letter symbol \(\emptyset\) or by empty braces \(\{\}\).

Examples:
  • \(A = \{ x \in \mathbb{R} \mid x^2 + 1 = 0 \}\) (Since no real number squared is negative, \(A = \emptyset\).)
  • \(B = \{ x \in \mathbb{N} \mid 1 < x < 2 \}\) (No natural number exists between 1 and 2, so \(B = \emptyset\).)
CRITICAL WARNING: The notation \(\{\emptyset\}\) does not represent an empty set. It represents a singleton set whose sole element is the empty set itself. Thus, \(n(\emptyset) = 0\) but \(n(\{\emptyset\}) = 1\).

2. Singleton Set

A set containing exactly one element is called a singleton set.

Examples:
  • \(S = \{ x \in \mathbb{Z} \mid x + 5 = 5 \} = \{0\}\).
  • \(T = \{ x \in \mathbb{N} \mid x \text{ is even and prime} \} = \{2\}\).

3. Finite and Infinite Sets

  • Finite Set: A set is finite if it is either empty or contains a definite, countable number of elements that can be fully listed. The number of unique elements in a finite set \(A\) is called its cardinal number, denoted by \(n(A)\).
  • Infinite Set: A set whose elements cannot be fully listed because it contains an endless progression of members. Examples include \(\mathbb{N}\), \(\mathbb{Z}\), and any continuous interval like \((0, 1)\) on the real line.

4. Equal Sets vs. Equivalent Sets

  • Equal Sets (\(A = B\)): Two sets \(A\) and \(B\) are equal if they have exactly the same elements. \[A = B \iff (\forall x, x \in A \iff x \in B)\] Order and repetition do not affect equality. For example, \(\{1, 2\} = \{2, 1, 1\}\).
  • Equivalent Sets: Two finite sets \(A\) and \(B\) are equivalent if they have the same number of elements (same cardinality), even if the elements themselves are completely different. \[A \approx B \iff n(A) = n(B)\] For example, \(\{1, 2, 3\}\) and \(\{a, b, c\}\) are equivalent since both have cardinality 3, but they are not equal.

    Rule: All equal sets are equivalent, but not all equivalent sets are equal.

Subsets and Proper Subsets

A set can contain other sets as sub-structures. This relationship forms the basis of all mathematical hierarchies.

1. Definition of a Subset (\(A \subseteq B\))

A set \(A\) is said to be a subset of set \(B\) if every element of \(A\) is also an element of \(B\). \[A \subseteq B \iff (\forall x, x \in A \implies x \in B)\] If \(A\) is not a subset of \(B\), we write \(A \not\subseteq B\).

2. Proper Subset (\(A \subset B\))

If \(A \subseteq B\) and \(A \neq B\), then \(A\) is called a proper subset of \(B\), and \(B\) is called the superset of \(A\).

This means \(B\) contains all elements of \(A\), plus at least one additional element that does not belong to \(A\).

Core Mathematical Properties of Subsets

  1. Every set is a subset of itself: For any set \(A\), \(A \subseteq A\). (Reflexive property)
  2. The empty set is a subset of every set: For any set \(A\), \(\emptyset \subseteq A\). Proof: The statement \(\forall x, x \in \emptyset \implies x \in A\) is vacuously true because there exists no element \(x\) in \(\emptyset\) to make the hypothesis false.
  3. Proof of Equality: Two sets are equal if and only if they are subsets of each other. \[A = B \iff A \subseteq B \text{ and } B \subseteq A\] This "double containment" method is the standard algebraic way to prove set equations in advanced math.
  4. Total Number of Subsets: If a finite set \(A\) contains \(n\) elements, the total number of subsets it can have is: \[\text{Total Subsets} = 2^n\] 5. Total Number of Proper Subsets: Since a proper subset must exclude the set itself, the total number of proper subsets is: \[\text{Total Proper Subsets} = 2^n - 1\]

Intervals as Subsets of \(\mathbb{R}\)

Because the set of Real Numbers \(\mathbb{R}\) is continuous, we cannot list subsets in Roster form. Instead, we express real subsets using interval notation. Let \(a, b \in \mathbb{R}\) with \(a < b\):

  • Closed Interval: \([a, b] = \{ x \in \mathbb{R} \mid a \le x \le b \}\) (Includes boundary points)
  • Open Interval: \((a, b) = \{ x \in \mathbb{R} \mid a < x < b \}\) (Excludes boundary points)
  • Semi-Open / Semi-Closed: \([a, b) = \{ x \in \mathbb{R} \mid a \le x < b \}\) (Includes \(a\), excludes \(b\))

Power Set

The concept of constructing a set out of all subsets is highly powerful and forms the baseline for many advanced algebra and probability questions.

Definition of a Power Set (\(P(A)\))

The collection of all subsets of a set \(A\) is called the power set of \(A\), denoted by \(P(A)\). \[P(A) = \{ S \mid S \subseteq A \}\] Every element of the power set \(P(A)\) is itself a set!

Crucial Power Set Relations and Cardinality

Let \(A = \{1, 2\}\). Its subsets are \(\emptyset, \{1\}, \{2\}, \{1, 2\}\). Thus: \[P(A) = \{ \emptyset, \{1\}, \{2\}, \{1, 2\} \}\] Note the following logical truths for any set \(A\):
  • \(\emptyset \in P(A)\) (The empty set is an element of the power set)
  • \(\emptyset \subseteq P(A)\) (The empty set is a subset of the power set, which is true for *any* set)
  • \(A \in P(A)\) (The set \(A\) itself is an element of its power set)
  • If \(n(A) = m\), then the cardinality of the power set is: \[n(P(A)) = 2^m\]

Power Set of the Empty Set

What is the power set of the empty set \(\emptyset\)? Since \(\emptyset\) has \(0\) elements, its only subset is \(\emptyset\) itself. \[P(\emptyset) = \{\emptyset\}\] Therefore, the cardinality is \(n(P(\emptyset)) = 2^0 = 1\).

If we take the power set of that: \[P(P(\emptyset)) = P(\{\emptyset\}) = \{ \emptyset, \{\emptyset\} \}\] Its cardinality is \(n(P(P(\emptyset))) = 2^1 = 2\). This progression forms the basis of constructing number systems from pure logic!

Universal Set

In any mathematical discussion or problem, all sets under consideration are treated as subsets of a larger, baseline set. This baseline set is called the Universal Set, denoted by \(U\).

The choice of a universal set depends entirely on the context:

  • In coordinate geometry, the universal set is the set of all points in the Cartesian plane \(\mathbb{R}^2\).
  • In real analysis or high school algebra, the universal set is usually the set of real numbers \(\mathbb{R}\).
  • If we are studying the properties of integers, the universal set is \(\mathbb{Z}\).

Solved Examples

Example 1: Deciphering Elements vs. Subsets

Problem: Let \(A = \{ 1, 2, \{3, 4\}, 5 \}\). Which of the following statements are incorrect and why?
(i) \(\{3, 4\} \subseteq A\)
(ii) \(\{3, 4\} \in A\)
(iii) \(\{\{3, 4\}\} \subseteq A\)
(iv) \(\{1, 2, 5\} \subset A\)
Solution: Let's analyze the elements of set \(A\). Set \(A\) contains exactly four elements: the number \(1\), the number \(2\), the set \(\{3, 4\}\), and the number \(5\).
  1. Statement (i) is INCORRECT. The entity \(\{3, 4\}\) is an element of \(A\), not a subset. For \(\{3, 4\}\) to be a subset, its individual elements \(3\) and \(4\) would have to be elements of \(A\). However, \(3 \notin A\) and \(4 \notin A\) (they are locked inside the nested set).
  2. Statement (ii) is CORRECT. The set \(\{3, 4\}\) is indeed listed directly as a member of \(A\), so \(\{3, 4\} \in A\).
  3. Statement (iii) is CORRECT. Since \(\{3, 4\} \in A\), if we wrap it in another set, we get \(\{\{3, 4\}\}\). Every element of this new set (which is the element \(\{3, 4\}\)) is a member of \(A\). Hence, \(\{\{3, 4\}\} \subseteq A\).
  4. Statement (iv) is CORRECT. The elements \(1, 2,\) and \(5\) are all members of \(A\), and the subset \(\{1, 2, 5\}\) is not equal to \(A\) (since it lacks \(\{3, 4\}\)). Therefore, it is a proper subset: \(\{1, 2, 5\} \subset A\).

Example 2: Iterating Power Sets

Problem: Find the cardinality of \(P(P(P(\emptyset)))\).
Solution: Let's work outward from the innermost set:
  1. Let \(A_0 = \emptyset\). We know that: \[n(A_0) = 0\]
  2. Let \(A_1 = P(A_0) = P(\emptyset)\). The cardinality of the power set is: \[n(A_1) = 2^{n(A_0)} = 2^0 = 1\] Indeed, \(A_1 = \{\emptyset\}\).
  3. Let \(A_2 = P(A_1) = P(P(\emptyset))\). The cardinality is: \[n(A_2) = 2^{n(A_1)} = 2^1 = 2\] Indeed, \(A_2 = \{\emptyset, \{\emptyset\}\}\).
  4. Let \(A_3 = P(A_2) = P(P(P(\emptyset)))\). The cardinality is: \[n(A_3) = 2^{n(A_2)} = 2^2 = 4\] The subsets of \(\{\emptyset, \{\emptyset\}\}\) are: \(\emptyset\), \(\{\emptyset\}\), \(\{\{\emptyset\}\}\), and \(\{\emptyset, \{\emptyset\}\}\). Thus, \(P(P(P(\emptyset))) = \{ \emptyset, \{\emptyset\}, \{\{\emptyset\}\}, \{\emptyset, \{\emptyset\}\} \}\).
Thus, the cardinality of \(P(P(P(\emptyset)))\) is **4**.

Example 3: Counting Subsets under Restrictions

Problem: Let \(A = \{1, 2, 3, 4, 5, 6, 7, 8\}\). Find:
(i) The number of subsets of \(A\) that contain the element \(1\).
(ii) The number of subsets of \(A\) that contain the element \(1\) but exclude the element \(8\).
Solution:
  1. Part (i): A subset of \(A\) can either contain or not contain any particular element. The set \(A\) has \(8\) elements. If a subset must contain the element \(1\), we have no choice for the element \(1\) (1 choice: it must be in the subset). For the remaining \(7\) elements, we have \(2\) choices for each (either in or out).

    Thus, the number of such subsets is: \[1 \cdot 2^7 = 128\] Alternatively, think of it as taking any of the \(2^7\) subsets of the set \(\{2, 3, 4, 5, 6, 7, 8\}\) and inserting the element \(1\) into all of them. This gives exactly \(128\) subsets.
  2. Part (ii): If the subset must contain \(1\) and must not contain \(8\), then both elements are fixed (1 choice each). We are left with \(6\) free elements \(\{2, 3, 4, 5, 6, 7\}\) which can either be included or excluded.

    Thus, the number of such subsets is: \[2^6 = 64\]
