Number System

Classification, standard sets and interval constraints

What is a Number?

A number is an abstraction — a mental construct decoupled from physical reality.

Consider observing two apples and two oranges. The apples possess physical, intrinsic properties: they are red, juicy, spherical and organic. The oranges are orange, acidic and textured.

Neither the apples nor the oranges contain a physical property called "two." You can slice an apple to find its seeds, but you can never dissect an object to isolate its "twoness."

The number \(2\) is an abstracted commonality. It is an invariant property extracted by the human mind when comparing distinct collections of objects. It ignores "what" the objects are and focuses entirely on "how many" there are.

The Transition to Pure Mathematics


In everyday life, we use numbers as adjectives to qualify physical things (e.g., "two apples").
However, mathematics begins the exact moment we remove the noun.
By isolating the abstraction \('2'\) from the physical objects, it ceases to be a descriptor and becomes an independent object of thought.
Once numbers exist purely as abstractions, we can define rules to manipulate them — such as addition (\(2 + 3 = 5\)) — independent of whether we are adding apples, planets, or vectors.
Understanding this is vital: algebra is simply the next layer of abstraction.
If a number abstracts away the nature of the object, a variable (\(x\)) abstracts away the value of the number itself.

The Real Number Line \(\mathbb{R}\)

In classical algebra and calculus, our universe of discourse is primarily restricted to the set of Real Numbers \(\mathbb{R}\). Every real number corresponds uniquely to a geometric point on an infinite continuous line.

For JEE problem-solving, we partition this continuous space into highly structured, discrete and continuous subsets. Understanding these boundaries is critical for defining the domains of functions.

Hierarchy of Number Sets

To avoid ambiguity in question statements (e.g., "where \(x\) is a positive integer" vs "where \(x\) is non-negative"), we utilize standard mathematical set notations:

Natural Numbers \(\mathbb{N}\)

The counting numbers: \(\{1, 2, 3, \dots\}\). Note that \(0 \notin \mathbb{N}\).

Integers \(\mathbb{Z}\) or \(\mathbb{I}\)

Complete values including negative, zero, and positive: \(\{\dots, -2, -1, 0, 1, 2, \dots\}\).

  • Non-negative integers: \(\{0, 1, 2, \dots\}\) (Includes zero)
  • Positive integers: \(\{1, 2, 3, \dots\}\) (Identical to \(\mathbb{N}\))

Rational Numbers \(\mathbb{Q}\)

Numbers expressible in the form \(\frac{p}{q}\), where \(p, q \in \mathbb{Z}\) and the denominator \(q \neq 0\). Their decimal representations either terminate or repeat periodically.

Irrational Numbers \(\mathbb{Q}^c\) or \(\mathbb{Q}'\)

Real numbers that cannot be represented as a ratio of two integers (e.g., \(\sqrt{2}\), \(\pi\), \(e\)). Their decimal expansions are non-terminating and non-recurring.

Rational and Irrational Numbers together constitute Real Numbers.

Complex Numbers \(\mathbb{C}\) encompass all real numbers as well as imaginary numbers.

A complex number can be written as \(a + ib\) where \(a, b \in \mathbb{R}\). An imaginary number necessarily contains a non-zero imaginary component (\(b \neq 0\)).

Continuous Subsets & Interval Notation

Unlike discrete elements which can be listed inside curly braces \(\{a, b\}\), continuous segments of the real number line are expressed via inequalities and intervals. This is the primary language used to express algebraic solution sets.

Notation Type Mathematical Inequality Interval Representation Geometric Meaning
Closed Interval \(a \le x \le b\) \(x \in [a, b]\) Includes both boundary points \(a\) and \(b\).
Open Interval \(a < x < b\) \(x \in (a, b)\) Excludes both boundary points \(a\) and \(b\).
Semi-Open / Semi-Closed \(a \le x < b\) \(x \in [a, b)\) Includes \(a\), excludes \(b\).
Infinite Interval \(x \ge a\) \(x \in [a, \infty)\) All values from \(a\) extending to positive infinity. Infinity (\(\infty\)) is always paired with an open bracket.

The symbol \(\infty\) (or \(-\infty\)) is not a real number, as it does not possess a definite, fixed value on the real number line.

Instead, it is a operational symbol indicating that a value grows boundlessly in the positive (or negative) direction without termination.

In mathematical definitions, an entity is classified as a specific number only if it represents a unique, definite value.

Because \(\infty\) cannot be pinned down to a fixed point, it can never be reached or included, which is why infinite boundaries are strictly assigned open brackets.
