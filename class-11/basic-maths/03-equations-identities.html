Equations vs Identities

The logical distinction, the Identity Theorem for polynomials, and conditional statements

Equations vs Identities: The Logical Difference

In mathematics, a statement of equality can represent two fundamentally different concepts depending on the set of values that satisfy it. Distinguishing between these two is critical for solving algebra problems correctly.

1. Equation

An equation is a statement of equality between two mathematical expressions that holds true only for specific values of the variable(s) involved. These specific values are called the roots or solutions of the equation.

Example: \[x^2 - 5x + 6 = 0\] This equation is true only when \(x = 2\) or \(x = 3\). For any other real number, the equality is false.

2. Identity

An identity is a statement of equality between two mathematical expressions that holds true for all values of the variable(s) for which the expressions are defined (i.e., within their common domain).

Example: \[(x - 2)(x - 3) = x^2 - 5x + 6\] This is true for every real number \(x\). If you substitute \(x = 0\), \(x = -5\), or \(x = 1000\), both sides will evaluate to the exact same value.

Key Comparison

  • An equation is a constraint. It asks: "For what specific values is this statement true?"
  • An identity is a transformation. It states: "These two expressions are different ways of writing the exact same mathematical entity."
  • Trigonometric identities (like \(\sin^2 x + \cos^2 x = 1\)) are identities because they hold for all real \(x\), whereas \(\sin x = \frac{1}{2}\) is an equation because it is satisfied only by discrete values (\(x = \frac{\pi}{6}, \frac{5\pi}{6}, \dots\)).

The Identity Theorem for Polynomials

For polynomial expressions, the boundary between being an equation and an identity is mathematically rigid. This relationship is defined by a fundamental theorem in algebra.

The Identity Theorem

Consider a polynomial equation of degree \(n\): \[P(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0 = 0\] where \(a_n, a_{n-1}, \dots, a_0\) are real or complex coefficients.
  1. If \(P(x) = 0\) is a polynomial equation of degree \(n\), it can have at most \(n\) distinct roots.
  2. If the equation is satisfied by more than \(n\) distinct values of \(x\), then the polynomial must be the zero polynomial. That is, it is an identity, and all its coefficients must be zero: \[a_n = a_{n-1} = \dots = a_1 = a_0 = 0\]

Why the Theorem is an Absolute Cheat-Code for JEE

If you have a quadratic equation (degree 2) like \(Ax^2 + Bx + C = 0\), and a question states it is satisfied by three distinct values of \(x\), you do not need to solve the quadratic. By the Identity Theorem, you can immediately conclude: \[A = 0, \quad B = 0, \quad C = 0\] This allows you to convert a difficult polynomial problem into a simple system of linear equations!

Solved Example: Proving a Complex Identity

Problem: Show that the following equation is an identity for all real numbers \(x\): \[\frac{(x-a)(x-b)}{(c-a)(c-b)} + \frac{(x-b)(x-c)}{(a-b)(a-c)} + \frac{(x-c)(x-a)}{(b-c)(b-a)} = 1\] where \(a, b, c\) are distinct real constants.
Solution: Let's analyze the expression. The left-hand side is a sum of three terms, where each term is a quadratic expression in \(x\) (since the numerator of each term is a product of two linear factors of the form \((x-\alpha)(x-\beta)\)).

Therefore, the entire left-hand side is a polynomial of degree **at most 2**. Let \(P(x)\) be the polynomial obtained by subtracting \(1\) from the left-hand side: \[P(x) = \frac{(x-a)(x-b)}{(c-a)(c-b)} + \frac{(x-b)(x-c)}{(a-b)(a-c)} + \frac{(x-c)(x-a)}{(b-c)(b-a)} - 1\] Since \(P(x)\) is a polynomial of degree \(\le 2\), it can have at most 2 distinct roots, unless it is an identity. Let's evaluate \(P(x)\) at three distinct test points: \(x = a\), \(x = b\), and \(x = c\):
  1. Evaluate at \(x = a\): \[P(a) = \frac{(a-a)(a-b)}{(c-a)(c-b)} + \frac{(a-b)(a-c)}{(a-b)(a-c)} + \frac{(a-c)(a-a)}{(b-c)(b-a)} - 1\] \[P(a) = 0 + 1 + 0 - 1 = 0\] Hence, \(x = a\) is a root.
  2. Evaluate at \(x = b\): \[P(b) = \frac{(b-a)(b-b)}{(c-a)(c-b)} + \frac{(b-b)(b-c)}{(a-b)(a-c)} + \frac{(b-c)(b-a)}{(b-c)(b-a)} - 1\] \[P(b) = 0 + 0 + 1 - 1 = 0\] Hence, \(x = b\) is a root.
  3. Evaluate at \(x = c\): \[P(c) = \frac{(c-a)(c-b)}{(c-a)(c-b)} + \frac{(c-b)(c-c)}{(a-b)(a-c)} + \frac{(c-c)(c-a)}{(b-c)(b-a)} - 1\] \[P(c) = 1 + 0 + 0 - 1 = 0\] Hence, \(x = c\) is a root.
We have shown that the polynomial \(P(x)\) of degree \(\le 2\) is satisfied by **three distinct roots** (\(a, b, c\)). By the Identity Theorem, \(P(x)\) must be identically zero! \[P(x) \equiv 0 \implies \frac{(x-a)(x-b)}{(c-a)(c-b)} + \frac{(x-b)(x-c)}{(a-b)(a-c)} + \frac{(x-c)(x-a)}{(b-c)(b-a)} \equiv 1\] This algebraic equality is an identity, satisfied for all \(x \in \mathbb{R}\).

Conditional Identities

A **conditional identity** is a mathematical relation that behaves as an identity (satisfied by all variables in the domain) but ONLY when a specific **auxiliary constraint** is satisfied.

The Classical Tri-Variable Constraint

The most famous conditional identities in JEE revolve around the constraint \(a + b + c = 0\).

If \(a + b + c = 0\), then:

  1. \(a^3 + b^3 + c^3 = 3abc\)
  2. \(a^2 + b^2 + c^2 = -2(ab + bc + ca)\)

Solved Example: Factoring using Conditional Identities

Problem: Factorize the expression: \[S = (x-y)^3 + (y-z)^3 + (z-x)^3\]
Solution: Let's define three variables: \[a = x - y, \quad b = y - z, \quad c = z - x\] First, check the sum of these variables: \[a + b + c = (x - y) + (y - z) + (z - x) = x - x + y - y + z - z = 0\] Since \(a + b + c = 0\), we can immediately apply the conditional identity: \[a^3 + b^3 + c^3 = 3abc\] Substitute the original expressions back: \[(x - y)^3 + (y - z)^3 + (z - x)^3 = 3(x - y)(y - z)(z - x)\] This instantly factorizes a highly complicated algebraic expression without any tedious expansions!
