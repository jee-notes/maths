Basic Functions & Domain

Introducing functions, domain boundaries, modulus, greatest integer, and fractional part operations

What is a Function?

A function \(f\) is a rule of mapping from a set \(A\) (the **domain**) to a set \(B\) (the **codomain**) that assigns to each element \(x \in A\) exactly one unique element \(y \in B\). We write this as: \[f: A \rightarrow B\] where \(y = f(x)\) is the image of \(x\) under \(f\).

Domain and Range

  • Domain: The set of all possible real input values \(x\) for which the function \(f(x)\) is mathematically defined and yields a real number output.
  • Range: The set of all actual output values \(y\) that the function produces as \(x\) varies over its domain.

Core Domain Constraints for Real Functions

To find the domain of any real-valued algebraic function, apply these two fundamental constraints:
  1. Denominator Constraint: Denominators can never be zero. For \(\frac{1}{g(x)}\), we require \(g(x) \neq 0\).
  2. Radical Constraint: Expressions under an even root (like square root) must be non-negative. For \(\sqrt{g(x)}\), we require \(g(x) \ge 0\).

The Modulus Function \(|x|\)

The modulus function (or absolute value function) represents the absolute magnitude of a real number, discarding its algebraic sign. Geometrically, \(|x|\) measures the positive distance of \(x\) from the origin \(0\) on the real number line.

Formal Piece-wise Definition

\[f(x) = |x| = \begin{cases} x & \text{if } x \ge 0 \\ -x & \text{if } x < 0 \end{cases}\] The domain of \(|x|\) is \(\mathbb{R}\), and its range is \([0, \infty)\).

The Famous Square-Root Trap

Many students believe that \(\sqrt{x^2} = x\). This is only true if \(x\) is positive. If \(x = -5\), then \(\sqrt{(-5)^2} = \sqrt{25} = 5\), which is \(-x\), not \(x\).

The correct algebraic definition is: \[\sqrt{x^2} \equiv |x|\] This identity is a frequent source of errors in calculus and coordinate geometry. Always write \(\sqrt{x^2}\) as \(|x|\)!

Modulus Properties

  1. \(|-x| = |x|\)
  2. \(|xy| = |x||y|\)
  3. \(\left|\frac{x}{y}\right| = \frac{|x|}{|y|} \quad (y \neq 0)\)
  4. \(|x + y| \le |x| + |y| \quad\) (Triangle Inequality: equality holds if \(xy \ge 0\))

The Greatest Integer Function (GIF) \([x]\)

The Greatest Integer Function, represented as \([x]\) (and also known as the **floor function**), outputs the greatest integer that is less than or equal to \(x\).

Formal Definition

For any real number \(x\), \([x] = n\), where \(n \in \mathbb{Z}\) is the unique integer such that: \[n \le x < n + 1\] The domain of \([x]\) is \(\mathbb{R}\), and its range is the set of Integers \(\mathbb{Z}\).

Key Examples (Notice Negative Values!)

  • \([3.7] = 3\)
  • \([5.0] = 5\)
  • \([0.8] = 0\)
  • \([-0.2] = -1 \quad\) (Since \(-1 \le -0.2 < 0\))
  • \([-4.9] = -5 \quad\) (Since \(-5 \le -4.9 < -4\))

Core Properties of GIF

  1. \([x] \le x < [x] + 1\)
  2. \(x - 1 < [x] \le x\)
  3. Integer Shift: \([x + k] = [x] + k \quad\) if and only if \(k \in \mathbb{Z}\).
  4. Symmetry: \[[x] + [-x] = \begin{cases} 0 & \text{if } x \in \mathbb{Z} \\ -1 & \text{if } x \notin \mathbb{Z} \end{cases}\]

The Fractional Part Function \(\{x\}\)

Every real number \(x\) can be broken down into its integer component and a fractional residue. The **fractional part function**, written as \(\{x\}\), isolates this fractional residue.

Formal Definition

\[\{x\} = x - [x]\] The domain of \(\{x\}\) is \(\mathbb{R}\), and its range is the semi-open interval \([0, 1)\).

Key Examples (Always Non-Negative!)

Since \(\{x\}\) is a fractional residue, it can **never** be negative:
  • \(\{4.3\} = 4.3 - [4.3] = 4.3 - 4 = 0.3\)
  • \(\{8.0\} = 8.0 - [8.0] = 8.0 - 8 = 0\)
  • \(\{-1.3\} = -1.3 - [-1.3] = -1.3 - (-2) = 0.7 \quad\) (Note: it is not \(-0.3\))

Core Properties of Fractional Part

  1. \(0 \le \{x\} < 1\)
  2. Periodic Nature: \(\{x + k\} = \{x\} \quad\) for any integer \(k \in \mathbb{Z}\).
  3. Symmetry: \[\{x\} + \{-x\} = \begin{cases} 0 & \text{if } x \in \mathbb{Z} \\ 1 & \text{if } x \notin \mathbb{Z} \end{cases}\]

JEE Solved Examples

Example 1: Domain of a GIF and Fractional Part Expression

Problem: Find the domain of the function: \[f(x) = \frac{1}{\sqrt{x - [x]}}\]
Solution: Using the radical and denominator constraints:
  1. For the square root in the denominator, the expression inside must be strictly positive: \[x - [x] > 0\]
  2. By the definition of the fractional part function: \[x - [x] \equiv \{x\}\]
  3. Thus, we require: \[\{x\} > 0\]
  4. We know that the range of \(\{x\}\) is \([0, 1)\). Therefore, \(\{x\}\) is always non-negative (\(\{x\} \ge 0\)).
  5. The only points where \(\{x\} > 0\) is violated is when \(\{x\} = 0\).
  6. The fractional part \(\{x\}\) is zero if and only if \(x\) is an integer (\(x \in \mathbb{Z}\)).
Therefore, the function is defined for all real numbers except integers.

Domain: \(x \in \mathbb{R} \setminus \mathbb{Z}\)

Example 2: Solving a Mixed Equation

Problem: Solve the equation for real \(x\): \[2[x] = x + \{x\}\]
Solution: Recall the fundamental relationship: \(x = [x] + \{x\}\). We substitute this representation of \(x\) into the right-hand side of the equation: \[2[x] = ([x] + \{x\}) + \{x\}\] Combine like terms: \[2[x] = [x] + 2\{x\}\] Subtract \([x]\) from both sides: \[[x] = 2\{x\}\] Since \([x]\) is by definition an integer (\([x] \in \mathbb{Z}\)), the right-hand side \(2\{x\}\) must also be an integer. Let's call this integer \(n\): \[[x] = n \quad \text{and} \quad 2\{x\} = n \implies \{x\} = \frac{n}{2} \quad (n \in \mathbb{Z})\] Now, apply the fundamental constraint on the fractional part: \[0 \le \{x\} < 1\] Substitute \(\{x\} = \frac{n}{2}\) into this inequality: \[0 \le \frac{n}{2} < 1\] Multiply by \(2\): \[0 \le n < 2\] Since \(n\) must be an integer, the only possible values for \(n\) are: \[n = 0 \quad \text{or} \quad n = 1\] Let's solve for \(x\) in each case:
  • Case 1: \(n = 0\) \[[x] = 0 \quad \text{and} \quad \{x\} = \frac{0}{2} = 0\] \[x = [x] + \{x\} = 0 + 0 = 0\]
  • Case 2: \(n = 1\) \[[x] = 1 \quad \text{and} \quad \{x\} = \frac{1}{2} = 0.5\] \[x = [x] + \{x\} = 1 + 0.5 = 1.5\]
Solutions: \(x = 0\) and \(x = 1.5\)
